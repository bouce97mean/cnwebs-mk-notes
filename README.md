# cnwebs-mk-notes

本仓库用于归档和发布一系列独立 HTML 页面。这些页面主要记录与网络信息相关的个人笔记、资料整理或工具备份，不针对任何特定网站或服务。

## 项目简介

这是一个静态页面集合，通过 HTML 文件的形式保存和展示各类笔记内容。仓库本身不包含后端服务或数据库，所有页面均为纯前端结构，可直接在浏览器中打开查看。适合用于长期保存和快速查阅。

## 目录说明

```
/
├── index.html          # 项目入口页面，可链接到其他归档页面
├── pages/              # 主要页面归档目录
│   ├── example-1.html  # 示例页面（实际内容请参考具体文件）
│   └── example-2.html
├── assets/             # 静态资源（如 CSS、JavaScript、图片等）
│   ├── style.css
│   └── script.js
└── README.md           # 本文件
```

- `pages/`：存放所有独立 HTML 页面，每页对应一个主题或笔记。
- `assets/`：包含页面公用的样式、脚本及图片资源。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可单独打开使用。
- 页面内容以笔记、资料整理或工具界面为主，不包含实时数据或动态服务。
- 归档的页面可能不定期更新或补充，不保证所有页面长期保持一致风格。
- 部分页面可能引用外部资源（如字体、图标库），但不会收集用户信息或执行追踪。
